# Chronic Kidney Disease Classification Using Machine Learning

## Project Overview

This project aims to classify Chronic Kidney Disease (CKD) using Machine Learning techniques based on patient clinical attributes.

## Dataset

The dataset contains 400 patient records with medical parameters such as:

- Age
- Blood Pressure
- Albumin
- Sugar
- Hemoglobin
- Serum Creatinine
- Packed Cell Volume
- White Blood Cell Count
- Red Blood Cell Count

## Machine Learning Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Specificity
- Confusion Matrix

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 98.75% |
| Decision Tree | 98.75% |
| Random Forest | 100% |

## Final Model

Random Forest was selected as the final prediction model and saved as:

```text
ckd_model.pkl
```
